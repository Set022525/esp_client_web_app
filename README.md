## 概要
Chrome などの Web Bluetooth 対応ブラウザから ESP32 BLE サーバに接続し、スライダー操作と方向ボタンで L6471 ステッピングモータを制御するフロントエンドです。積算ステップ数と回転数をリアルタイム表示します。

## 必要環境
- Web Bluetooth 対応ブラウザ (Chrome 推奨)
- HTTPS または localhost で配信されたページ
- ESP32 側の BLE サービス/キャラクタリスティック UUID が下記と一致していること
  - Service: `12345678-1234-5678-1234-56789abcdef0`
  - Command Characteristic: `12345678-1234-5678-1234-56789abcdef1`
  - Position Characteristic: `12345678-1234-5678-1234-56789abcdef2`
  - デバイス名プレフィックス: `ESP32-L6471`

## 使い方
1. ブラウザで `index.html` を開く。
2. 「ESP32に接続」を押し、`ESP32-L6471` デバイスを選択。
3. 接続後に表示されるスライダーで 0–100 の速度を選択。
4. 正転(🔼) / 逆転(🔽) ボタンを押している間だけ回転します。離すと停止(0送信)。
5. 位置キャラクタリスティックからの通知を購読し、積算ステップと推定回転数を表示します（1/8 μステップ・200step/rev 前提）。

## ファイル
- `index.html`: UI 本体
- `js/main.js`: BLE 接続、方向ボタンの押下送信、位置通知の購読と表示
- `css/style.css`: 簡易スタイル (必要に応じて調整)

## 開発メモ
- UUID/デバイス名を変更した場合は `js/main.js` 冒頭の定数を合わせてください。
- コマンド送信形式は 1 バイト: `0`=停止、`1–100`=正転、`0x80|(1–100)`=逆転。
- 位置は Little Endian の int32 (microstep カウント) で通知されます。換算係数を変える場合は `MICROSTEPS_PER_REV` を調整。
