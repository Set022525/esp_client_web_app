## 概要
Chrome などの Web Bluetooth 対応ブラウザから ESP32 BLE サーバに接続し、スライダー操作で L6471 ステッピングモータの速度値 (0–100) を 1 バイトで送信する簡易フロントエンドです。

## 必要環境
- Web Bluetooth 対応ブラウザ (Chrome 推奨)
- HTTPS または localhost で配信されたページ
- ESP32 側の BLE サービス/キャラクタリスティック UUID が下記と一致していること
  - Service: `12345678-1234-5678-1234-56789abcdef0`
  - Characteristic: `12345678-1234-5678-1234-56789abcdef1`
  - デバイス名プレフィックス: `ESP32-L6471`

## 使い方
1. ブラウザで `index.html` を開く。
2. 「ESP32に接続」を押し、`ESP32-L6471` デバイスを選択。
3. 接続後に表示されるスライダーで 0–100 の値を選び、「値を送信」を押すと BLE write で ESP32 に 1 バイト送信されます。
4. 0 を送信すると ESP32 側でモータをソフトストップします。

## ファイル
- `index.html`: UI 本体
- `js/main.js`: BLE 接続と値送信ロジック
- `css/style.css`: 簡易スタイル (必要に応じて調整)

## 開発メモ
- UUID/デバイス名を変更した場合は `js/main.js` 冒頭の定数を合わせてください。
- 値の送信形式は `Uint8Array([value])` の 1 バイトです。ESP32 側の解釈ロジックに合わせてください。
